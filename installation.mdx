## Requirements

Semgrep requires **Python 3.10 or later**. It runs on macOS, Linux, and Windows (via WSL).

<Tip>
  Give Semgrep its own virtual environment to avoid dependency conflicts with other Python projects.
</Tip>

## Install Semgrep

<CodeGroup>

```bash pip (recommended)
python3 -m pip install semgrep
```

```bash Homebrew (macOS)
brew install semgrep
```

```bash Docker
docker pull semgrep/semgrep:latest
```

```bash conda
conda install -c conda-forge semgrep
```

</CodeGroup>

## Verify the installation

After installing, confirm Semgrep is working:

```bash
semgrep --version
```

You should see output like:

```
1.155.0
```

## Platform notes

### pip

pip installs Semgrep into the active Python environment. To keep your system Python clean, use a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install semgrep
```

On Windows, run Semgrep inside WSL (Windows Subsystem for Linux). Native Windows support is not available.

### Homebrew

Homebrew is the easiest installation method on macOS. Homebrew manages the Python dependency automatically.

### Docker

Docker lets you run Semgrep without any local Python installation. Mount your source directory into the container:

```bash
docker run --rm -v "${PWD}:/src" semgrep/semgrep semgrep scan --config=auto /src
```

To use the Semgrep AppSec Platform from Docker, pass your API token as an environment variable:

```bash
docker run -e SEMGREP_APP_TOKEN=<TOKEN> --rm -v "${PWD}:/src" semgrep/semgrep semgrep ci
```

<Note>
  Use `semgrep/semgrep:latest` to always pull the most recent release, or pin to a specific version tag such as `semgrep/semgrep:1.155.0` for reproducible builds.
</Note>

## Upgrading

Run the upgrade command that matches your installation method:

<CodeGroup>

```bash pip
python3 -m pip install --upgrade semgrep
```

```bash Homebrew (macOS)
brew upgrade semgrep
```

```bash Docker
docker pull semgrep/semgrep:latest
```

```bash conda
conda update -c conda-forge semgrep
```

</CodeGroup>

## Optional: log in to Semgrep

Logging in is optional but unlocks additional capabilities:

```bash
semgrep login
```

Logging in gives you access to:

- **Semgrep Supply Chain** — dependency scanning for reachable vulnerabilities in third-party libraries
- **Pro rules** — 600+ high-confidence rules written by Semgrep's security research team
- **Pro engine** — cross-file and cross-function analysis for detecting complex vulnerabilities

<Note>
  Code is never uploaded when you log in. Login only enables access to additional rules and the AppSec Platform dashboard.
</Note>
